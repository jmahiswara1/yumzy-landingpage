# Yumzy

Landing page interaktif dan modern untuk **Yumzy**, sebuah produk camilan premium (artisanal snacks) yang menonjolkan tekstur lembut, bahan sehat, dan varian rasa yang unik.

**Live Preview:** [eatyumzy.vercel.app](https://eatyumzy.vercel.app)

## Fitur & Teknologi

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Bahasa:** TypeScript
- **Animasi & UI:** Modern micro-animations & custom reusable components
